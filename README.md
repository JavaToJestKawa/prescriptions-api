# Prescriptions API

REST API implementation for managing prescriptions using ASP.NET Core and Entity Framework Core.

## Features
* Prescription management
* Database integration with Entity Framework Core
* Layered architecture with controllers, services, and DTOs
* Data persistence with migrations

## Technologies
* C#
* ASP.NET Core
* Entity Framework Core
* SQL Server

## Architecture
* Controllers
* Services
* DTOs
* Models
* Data layer
* Migrations

## Approach
* Code First

## How to run

dotnet restore  
dotnet run
